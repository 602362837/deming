## :rocket: 简介

在信息安全已经成为战略重点的背景下，组织不仅需要实施保护措施，还需要证明这些措施的有效性和长期一致性。Deming 为此提供了一个强大的开源解决方案，用于按照 ISO/IEC 27001 标准管理信息安全管理体系（ISMS）。

Deming 由 CISO 为 CISO 打造，兼具完整的功能覆盖、清晰的架构，以及对真实运营需求的高度适应性。凭借完善的文档和活跃的社区，它正在关键环境中逐步成为重要参考。

Deming 因质量和影响力而受到认可，是用于 GRC 与 ISMS 管理的优秀开源工具。

### :question: Deming 是什么？

**Deming** 是一款强大且直观的工具，用于管理、规划、监控和报告安全措施的有效性。根据 ISO/IEC 27001:2013 第 9 章，**Deming** 可帮助你在满足严格标准要求的同时，确保安全措施适当且成比例。

### :dart: 为什么要监控？

定期监控和评估安全措施至关重要，因为它可以：

- 评估现有控制措施的有效性。
- 验证安全要求是否得到满足。
- 持续改进信息安全。
- 为决策提供准确数据。
- 说明改进信息安全管理体系（ISMS）的必要性。

**Deming** 提供了有效实现这些目标所需的工具。

### :chart_with_upwards_trend: 绩效评估

根据 ISO 27001 第 9.1 章，必须对安全绩效进行评估。**Deming** 会引导你完成这一过程，帮助你：

- 明确需要监控和测量的内容。
- 选择合适的方法，确保结果有效。
- 规划监控和测量的时间。
- 确定每项任务的负责人。
- 分析并评估结果。

### 🐳 Docker 安装

你可以使用 Docker 快速启动开发模式下的本地实例：

```bash
git clone https://github.com/sourcentis/deming.git
cd deming
cp .env.example .env
sed -i 's/DB_HOST=127.0.0.1/DB_HOST=mysql/' .env
docker compose up
```
